## 二、方案对比

| 方案 | 每月费用 | 国内访问速度 | 自定义域名 | HTTPS | 部署方式 |
|------|---------|-------------|-----------|-------|---------|
| **① Vercel** | 免费 | ❌ 慢（大陆无节点） | 支持 | 自动 | `git push` 自动部署 |
| **② Cloudflare Pages** | 免费 | ⚠️ 一般（香港节点） | 支持 | 自动 | `git push` 或 CLI |
| **③ 国内轻量云 + Nginx** | ≈ ¥30-60/月 | ✅ 最快 | 需备案 | 免费证书 | 手动上传 |
| **④ 腾讯云 EdgeOne Pages** | 免费额度充足 | ✅ 快（国内 CDN） | 需备案 | 支持 | `git push` 或 CLI |
| **⑤ 对象存储 + CDN（腾讯COS/阿里OSS）** | ≈ ¥10-30/月 | ✅ 最快 | 需备案 | 支持 | 手动上传 |

> ⚠️ **Vercel 在中国大陆没有 CDN 节点**，最近的节点在香港。实测国内访问延迟 300-800ms，高峰期掉包严重，`*.vercel.app` 域名还可能被干扰。**不适合以国内访问为主的项目。**

## 三、推荐方案

### 首选：方案 ④ 腾讯云 EdgeOne Pages（国内访问最优 + 免运维）

**费用**：免费额度充足
- 每月 50GB 流量 / 50万次请求（免费）
- 自动 HTTPS + 国内 CDN 加速
- 支持关联 GitHub 仓库自动部署

**操作步骤**：

1. 在 [EdgeOne Pages 控制台](https://console.cloud.tencent.com/edgeone/pages) 创建项目
2. 关联本仓库，或上传 `00_工作台/运营数据管理/app/` 目录
3. 构建命令留空（纯静态项目）
4. 部署完成后获得 `*.edgeone.app` 域名
5. （可选）绑定已备案的域名，自动签发 HTTPS 证书

---

### 快速尝鲜：方案 ② Cloudflare Pages

**费用**：免费

**特点**：Cloudflare 在香港有节点，国内延迟一般在 200-400ms，比 Vercel 稍好但不保证稳定。适合临时体验，不适合正式内部使用。

```bash
npx wrangler pages deploy "00_工作台/运营数据管理/app" --project-name=aiwei-gallery
```

---

### 稳定首选：方案 ③ 国内轻量云 + Nginx

**费用**：¥30-60/月

**说明**：国内服务器直连，延迟最低（<20ms），完全可控。需要域名备案（5-20 个工作日）。

| 服务商 | 配置 | 价格 |
|-------|------|------|
| 腾讯云轻量服务器 | 2核 2G 4Mbps | ¥30/月 |
| 阿里云轻量应用服务器 | 2核 2G 3Mbps | ¥34/月 |

**操作步骤**：

1. 购买服务器（选成都/重庆地域，离美术馆最近，延迟最低）
2. 安装 Nginx，上传前端文件
3. 配置 HTTPS，绑定域名
4. 后续更新仍通过 Git 管理，手动 `scp` 或写个简单脚本同步

## 四、域名 & 备案（国内方案）

| 步骤 | 说明 | 耗时 |
|------|------|------|
| 购买域名 | 阿里云/腾讯云/Cloudflare，约 ¥30-80/年 | 10 分钟 |
| 实名认证 | 域名所有者身份验证 | 1-2 个工作日 |
| ICP 备案 | 选国内服务器必须；个人备案即可，选「美术馆/展览」类目 | 5-20 个工作日 |
| 公安备案 | 部分省市要求，备案通过后 ICP 侧有指引 | 1-2 周 |
| DNS 解析 | 域名指向部署平台或云服务器 IP | 10 分钟 |

**不备案也能用**：
- 用 Vercel/Cloudflare Pages 提供的 `*.vercel.app` / `*.pages.dev` 二级域名
- 国内访问可能偶尔变慢，但日常使用无障碍
- ****推荐初期先用 `*.vercel.app` 二级域名上线，备案期间不影响使用

## 六、安全注意事项

| 风险 | 影响 | 建议 |
|------|------|------|
| anon key 暴露 | 任何人可读写数据库 | 当前内部使用可接受；后续需加上 IP 白名单或 RLS |
| 无用户认证 | 任意访问者都可录入/删除数据 | 项目后续计划已包含 P0 迭代 4 账号登录系统 |
