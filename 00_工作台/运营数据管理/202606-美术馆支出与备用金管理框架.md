# 202606 美术馆支出与备用金管理框架

更新时间：2026-06-09

本文档基于 `01_运营行政/2026美术馆销售统计.xlsx` 中的 `备用金` 工作表整理，用于单独管理美术馆支出与备用金。本文档不记录经营收入。

## 一、管理原则

- 收入与支出分开管理。
- 本文件只回答：美术馆花了多少钱、为什么花、谁经手、是否有发票和付款凭证、关联哪个项目。
- 经营收入另行记录在 `202606-美术馆经营收入管理表.xlsx` 和对应收入管理框架中。

## 二、原始工作表结构

原始来源：`2026美术馆销售统计.xlsx` 的 `备用金` 工作表。

当前字段包括：

| 字段 | 说明 |
|---|---|
| 序号 | 备用金或支出记录序号 |
| 日期 | 借入或贷出日期 |
| 借入 | 备用金借入金额 |
| 贷出 | 备用金支出金额 |
| 项目 | 支出归属项目 |
| 内容 | 支出具体内容 |
| 经手人 | 经办/经手人员 |
| 发票 | 是否有发票 |
| 付款凭证 | 是否有付款凭证 |
| 备注 | 补充说明 |

## 三、当前读取到的项目类型

| 项目类型 | 示例内容 |

| 可乐团建 | 干花信封、果壳风铃、茶歇、压茶饼 |

## 四、建议支出与备用金明细表

建议表名：`支出备用金填报`

| 字段 | 说明 |
|---|---|
| 日期 | 借入或支出日期 |
| 类型 | 备用金借入 / 备用金支出 |
| 项目 | 运营、耗材、展览、团建活动、工坊、画廊、场地、其他 |
| 支出类别 | 材料、茶歇、设备、人工、交通、通信、打印、运费、保洁、其他 |
| 内容 | 具体支出内容 |
| 借入金额 | 备用金借入金额 |
| 贷出金额 | 实际支出金额 |
| 余额 | 借入金额减贷出金额后的滚动余额 |
| 经手人 | 经办/经手人员 |
| 是否有发票 | 有 / 无 / 不需要 / 待补 |
| 是否有付款凭证 | 有 / 无 / 不需要 / 待补 |
| 关联活动/展览 | 如可口可乐企业团建、某个展览、某个工坊 |
| 备注 | 特殊说明 |
| 月份 | 从日期自动生成 |
| 凭证状态 | 自动提示是否需要补凭证 |
