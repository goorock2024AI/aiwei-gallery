## 四、对未来的参考

1. **数据迁移路线**：Supabase REST API → 本地 PostgreSQL（直接 SQL INSERT）
2. **JSONB 字段处理**：Supabase 返回空数组 `[]`，本地 API 的 `toSnake` 函数需要递归处理才能正确写入
3. **NUMERIC 类型兼容**：PostgreSQL 的 `pg` 库返回 NUMERIC 类型为字符串，前端需要数字
4. **HTTP 环境限制**：浏览器 `crypto.subtle.digest()` 只在 HTTPS 下可用，密码校验必须走服务端
