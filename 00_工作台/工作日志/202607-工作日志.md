### 推送异常记录（v1.3.1 commit `c637e30`）

**状态**：commit 已落地但 `git push origin main` **未成功**；本会话（7-19）**重新尝试全部失败**，origin/main 仍是 v1.3.0（`a5d47b4`）。

**异常现象（7-18 上次会话 + 7-19 本会话均实测）**：
- HTTPS push：`Recv failure: Connection was reset` / `Failed to connect to github.com port 443 after 21xxx ms: Could not connect to server`
- HTTPS fetch / ls-remote：同样 21s 连接失败
- 已试过（均失败）：放宽 `GIT_HTTP_LOW_SPEED_*` 阈值、嵌入 OAuth token 到 URL、sleep 60s 重试
- SSH 路线：**Permission denied (publickey)** —— GitHub 上未注册公钥（仅有 OAuth token）
- 关键对比：同期 `gh auth status` / `gh repo view` 仍可达 → 网络层选择性封堵 `https://github.com:443`，但 `api.github.com:443` 与 `git@github.com:22` 协议层未封死

**唯一可行路径（需用户在浏览器手动 30 秒）**：

```bash

cat ~/.ssh/id_rsa.pub
# → 形如 ssh-rsa AAAA...== goorock@xxx

# 2. 浏览器打开 https://github.com/settings/keys → "New SSH key"
#    Title: aiwei-gallery-本机
#    Key:   粘贴上述公钥
#    → 点击 "Add SSH key"

# 3. 回到本仓库，切 SSH URL + 推送
cd "D:/工作文档/00_进行中/艾维美术馆"
git remote set-url origin git@github.com:goorock2024AI/aiwei-gallery.git
git push origin main
# 预期：推送 c637e30 → main（1 commit）+ 随后推送异常记录 commit（见下方"待 push")
```
