# 202606 企业团建活动 - 可口可乐

更新时间：2026-06-09

## 基础信息

| 项目 | 内容 |
|---|---|
| 活动名称 | 可口可乐企业团建活动 |
| 活动日期 | 2026-06-11 |
| 活动时间 | 9:00-12:00 |
| 活动地点 | 艾维美术馆美学空间 |
| 客户 | 可口可乐 |
| 活动状态 | 筹备中 |
| 活动内容 | 果壳风铃制作、干花信封制作、普洱茶压茶饼体验、茶歇服务 |
| 参与方式 | 分组不轮转，各20人 |
| 茶歇内容 | 烘焙 + 果盘 |
| 茶歇预算 | 1000元，其中800元采购服务商，200元美术馆自购 |

## 活动目标

- 顺利完成可口可乐企业团建接待。
- 保障三项体验活动同时或分区有序开展。
- 提供符合预算的茶歇服务。
- 通过本次活动验证美学空间企业团建接待流程。

## 活动内容模块

### 手工制作果壳风铃

- 参与人数：20人。
- 形式：固定分组，不轮转。
- 所需准备：果壳风铃材料包、工具、桌面保护、成品包装。

### 制作干花信封

- 参与人数：20人。
- 形式：固定分组，不轮转。
- 所需准备：干花、信封材料、胶类工具、桌面保护、成品包装。

### 普洱茶压茶饼体验

- 参与人数：20人。
- 形式：固定分组，不轮转。
- 所需准备：普洱茶压制设备、茶叶、指导老师、操作台、安全提示、成品包装。

### 茶歇服务

- 内容：烘焙 + 果盘。
- 总预算：1000元。
- 采购分配：
  - 服务商采购：800元。
  - 美术馆自购：200元。

| 时间 | 环节 | 备注 |

## 风险与注意事项

- 普洱茶压茶饼体验涉及设备操作，需提前做好安全提示。
- 三组不轮转，需确保每组体验内容时长充足、人员配置独立。
- 茶歇预算分为服务商采购和美术馆自购，需分别留存凭证。
- 美学空间活动结束后需恢复原状，避免影响后续使用。
