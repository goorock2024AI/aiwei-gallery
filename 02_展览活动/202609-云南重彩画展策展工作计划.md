## 三、与云南重彩研究院合作沟通清单

第一次见面需要谈清楚以下事项：

### 3.1 合作模式

- [x] **双方联合主办** — 美术馆与云南重彩研究院并列署名
- [x] **研究院角色**：推荐艺术家和作品；美术馆负责策划主体，研究院提供建议和意见
- [x] **费用**：无需支付借展费；希望研究院能引荐商业合作机会
- [x] **协议**：无需签署书面协议

### 3.2 作品方面

- [x] **作品数量**：根据展厅面积和以往经验，目标 80-100 件
- [x] **材质与装裱**：以纸本为主；为节约成本，优先请研究院提供已装裱完成的作品
- [ ] **现有作品是否够用**：需与研究院当面确认数量和件数
- [ ] **补充作品**：如有需要，由研究院推荐特邀艺术家补充
- [x] **文字资料**：确定参展作品后，由艺术家提供简介和作品阐释
- [x] **遴选原则**：优先选择本地作品和已装裱好的作品，以减少运输和装裱成本

### 3.3 主题方向

- [ ] **"回顾与展望"方向**：需与研究院确认是否认可
- [ ] **研究院建议**：需询问研究院是否有自己的主题建议
- [x] **板块划分思路**：按时间线编排（如：经典重彩 → 当代探索 → 跨界融合）

### 3.4 资源支持

- [ ] **企业资源引荐**：需见面时明确提出需求，请研究院引荐商业合作机会
- [x] **开幕式嘉宾**：研究院可协助邀请艺术家代表、文化界人士
- [x] **过往素材**：无可复用的媒体报道或画册素材

### 3.5 时间节点

- [x] **作品清单确认时间**：研究院可在7月中旬前确认约80%的作品清单，剩余部分后续补齐
- [ ] **运输情况**：需确认是否有外地作品需要调运；已向研究院提出优先选择本地作品

## 六、策展流程参考（针对无经验者）

以下是第一次策展的标准流程，每步说明"做什么"和"为什么"：

### Step 1：定方向
与合作方沟通，确定展览的主题、规模、作品数量。这决定了后面所有工作的基础。

### Step 2：定内容
从作品库中挑选展品，组织成一个有逻辑的故事——观众走进来，能按一条清晰线索看完，而不是一堆画的随机陈列。

### Step 3：定空间
根据作品的尺寸和数量，设计展厅布局：哪里是入口、哪里是重点展区、观众怎么走、每面墙挂哪些画。

### Step 4：写文案
- **前言**：展览的开场白，告诉观众这个展览在讲什么（500字以内）
- **板块说明**：每个分区的引导文字（200字以内）
- **展签**：每件作品的作品名、作者、年代、尺寸、材质、简要说明

### Step 5：设计制作
海报、邀请函、展签、展览手册、导视系统——确保视觉统一，与展览调性匹配。

### Step 6：宣传推广
分三个阶段：
- **预热期**（开幕前2-3周）：发布预告、公布亮点、制造期待
- **爆发期**（开幕前后）：媒体通稿、社交平台刷屏
- **持续期**（展期中）：阶段性的内容更新、活动招募

### Step 7：布展
实物操作：作品挂上墙、灯光打到位、展签对齐、整体效果调整。建议预留2-3天。

### Step 8：开幕式
嘉宾签到→致辞→参观导览→自由交流。开幕式不仅是仪式，更是建立关系（媒体、赞助商、藏家）的机会。

### Step 9：展期运营
日常维护、票务管理、团体接待、活动组织。
