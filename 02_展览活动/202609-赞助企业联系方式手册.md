## 实用联系话术模板

### 给消费品品牌（嘉华/潘祥记/云南白药）
> "您好，我是艾维美术馆的，我们今年9月中旬到10月底有一个云南重彩画展，位置在西山森林公园内。展览正好覆盖国庆黄金周，会有大量游客流量。想看看贵品牌是否有兴趣做展期合作，比如现场体验区、联名伴手礼之类的形式。方便转接市场部吗？"

### 给银行（建行/农行/红塔银行）
> "您好，我是艾维美术馆的。我们9月有一个重彩画展，美术馆在西山风景区内，环境比较私密高雅。了解到贵行有高端客户答谢活动的需求，我们可以提供专场导览+艺术体验的场地和内容服务。方便转接私人银行部或市场部吗？"

### 给酒店（华邑/忽岚间·悦）
> "您好，我是艾维美术馆的，美术馆就在西山上。我们9月到10月有一个云南重彩画展，想跟贵酒店做一个住客互导合作——住客凭房卡可以免费或优惠看展，给酒店增加一个增值服务。方便转接市场部吗？"

### 给旅游平台（携程/美团）
> "您好，我想咨询景点门票上线的合作。我们美术馆9月中旬有一个新的云南重彩画特展，想上线到平台销售，也看看能不能打包进西山一日游的线路里。方便对接负责景点合作的同学吗？"

## 查找更多企业联系方式的渠道

| 渠道 | 用途 |
|------|------|
| **企查查 / 天眼查** | 查企业联系电话、邮箱 |
| **企业官网「联系我们」** | 最直接的官方渠道 |
| **招聘平台（BOSS直聘/智联）** | 查看HR电话，可转达合作意向 |
| **行业展会/商会** | 云南省工商联、昆明市文旅局可引荐 |
| **现有合作方引荐** | 研究院、已有合作企业介绍最有效 |
