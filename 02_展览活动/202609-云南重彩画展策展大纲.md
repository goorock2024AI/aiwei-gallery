## 二、策展理念

云南重彩画是云南美术史上具有独特地位的绘画样式。它以中国传统工笔重彩为根基，融合云南多民族文化的色彩语言与装饰美学，在二十世纪八十年代以来形成了鲜明的地域风格。

本次展览以 **"回顾与展望"** 为核心线索，不在于追求全貌式的文献梳理，而在于通过一批具有代表性的作品，呈现云南重彩画的发展脉络——它从哪里来、走过哪些路、未来可能去向何方。

展览希望达成的三个目标：
1. **致敬传统** — 呈现重彩画的技法根基与经典面貌
2. **看见当下** — 展示当代重彩画家的探索与突破
3. **打开可能性** — 引发观众对重彩画未来方向的思考与讨论

## 三、展览板块结构

按时间线与风格演变，初步划分为三个单元：

### 第一单元：经典与传承

| 内容 | 说明 |
|------|------|
| 时间跨度 | 约1980s-2000s |
| 核心看点 | 云南重彩画的奠基时期，呈现经典技法与民族题材的成熟范式 |
| 作品方向 | 传统工笔重彩、民族风情题材、代表性艺术家的经典作品 |
| 空间氛围 | 沉稳、庄重，以暖色调灯光为主 |

### 第二单元：拓界与探索

| 内容 | 说明 |
|------|------|
| 时间跨度 | 约2000s-2010s |
| 核心看点 | 重彩画的题材拓展与技法革新，融入当代审美意识 |
| 作品方向 | 突破传统题材，探索抽象表达、综合材料、当代叙事的作品 |
| 空间氛围 | 活跃、丰富，灯光层次多变 |

### 第三单元：新语与展望

| 内容 | 说明 |
|------|------|
| 时间跨度 | 约2010s至今及未来 |
| 核心看点 | 年轻一代重彩画家的视角，跨媒介、跨文化的融合尝试 |
| 作品方向 | 实验性作品、跨界合作、数字媒介与传统重彩的结合 |
| 空间氛围 | 明亮、开放，可融入互动或多媒体元素 |

> **说明**：三个板块的比例大致为 4:3:3（经典占稍大比重），具体根据研究院提供的作品清单调整。

## 五、展览体验设想

### 5.1 空间与动线

- **入口**：设置前言墙 + 一幅代表性大尺幅作品作为视觉焦点，建立第一印象
- **三个单元依次展开**：观众沿顺时针或逆时针单向流动，形成"过去→现在→未来"的观展节奏
- **出口/尾声**：设置留言区或互动区，收集观众反馈，同时放置衍生品展示

### 5.2 视觉与设计

- **主色调**：建议以重彩画中代表性的石绿、朱砂、金色为基调
- **展陈方式**：纸本作品建议使用统一规格画框，保证视觉整齐度
- **灯光**：重点作品重点照明，整体以暖光为主

### 5.3 配套活动（初步设想）

| 活动 | 说明 |
|------|------|
| 开幕式导览 | 邀请策展人或艺术家代表现场讲解 |
| 重彩画体验工坊 | 国庆期间面向公众的体验活动（可收费） |
| 企业专场导览 | 针对赞助企业及潜在客户的专场接待 |
| 讲座/对谈 | 邀请艺术家或学者举办1-2场公开讲座 |
